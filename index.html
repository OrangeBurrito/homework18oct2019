1. What is a semantic tag & why you should use them

Semantic tags are just HTML tags with a meaning

Why you should use them: Semantic tags make your code more human-readable. (and makes it clear to the browser the meaning of the page and the content)

2. How does the browser render HTML & CSS

  1. The browser sends a request to the website host server to fetch the .html file
  2. The browser converts the HTML elements to Javascript which form a tree called the DOM (document object model)
  3. After the browser creates the DOM it goes to the CSS and constructs a CSSOM (CSS Object Model) that is similar to the DOM
  4. A Render tree is constructed from the DOM & CSSOM that calculates the layout of each visible element and displays them on screen.
